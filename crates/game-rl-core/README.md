# game-rl-core

Core types and traits for the Game-RL protocol.

## Features

- Agent types and registration (EntityBehavior, GameMaster, ColonyManager, etc.)
- Observation schemas (structured, vector, custom)
- Action spaces (discrete, continuous, parameterized)
- Reward components and shaping
- Vision stream descriptors (RGB, depth, segmentation, flow)
- Game manifest and capabilities
- Protocol error types with JSON-RPC error codes
